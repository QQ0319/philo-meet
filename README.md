# 🌀 Philo Meet

哲學系專用輕量級聊天室，無需註冊登入，適合快速協作與討論。

## 🚀 使用方式

1. 安裝依賴
```bash
npm install
```

2. 啟動伺服器
```bash
npm start
```

3. 開啟瀏覽器前往：
```
http://localhost:3000
```

## 📁 結構說明

- `index.html`：主畫面
- `script.js`：聊天室邏輯
- `style.css`：樣式調整
- `server.js`：Node.js + Socket.io server
- `package.json`：依賴與啟動設定
